## 1.1 土地资源相关
- 土地权属编码（land_code）：唯一标识地块的编码，格式为「省份简称-序号」（如GD-001代表广东省第1块地块）；
- 土地类型（land_type）：分为耕地、林地、草地、建设用地、未利用地5类，是自然资源审计的核心分类维度；
- 使用状态（use_status）：
    - 合规使用：土地用途与审批用途一致，无违规行为；
    - 违规占用：未经审批或超出审批范围使用土地；
    - 闲置：审批后超过1年未开发利用的土地；
    - 待开发：已完成审批但尚未启动开发的土地；
- 审计状态（audit_status）：未审计、审计中、已审计、存在问题（仅审计发现违规的地块标记为“存在问题”）；
- 审批日期（approval_date）：土地用途/开发权限获得审批的日期，格式为YYYY-MM-DD。

## 1.2 违规用地相关
- 违规类型（violation_type）：
    - 非法占用：无审批手续占用土地；
    - 超审批面积：实际使用面积超出审批面积；
    - 改变用途：未经审批将土地用于非审批用途（如耕地改建设用地）；
    - 未批先建：未完成审批即开工建设；
- 整改状态（rectification_status）：未整改、整改中、已整改、无法整改（如违规建筑已建成无法拆除）；
- 处罚金额（penalty_amount）：违规用地行为的行政处罚金额，单位为元，无处罚则为NULL。

## 1.3 矿产资源相关
- 矿产类型（mineral_type）：煤炭、铁矿、稀土等常见矿产种类；
- 开采量（mining_volume）：实际开采矿产的数量，单位为万吨；
- 审批开采量（approval_volume）：自然资源部门审批的年度开采额度，单位为万吨；
- 超采量：开采量 - 审批开采量（正数代表超审批开采，负数代表未达审批量）；
- 审计意见（audit_opinion）：审计人员对矿产开采行为的合规性评价。
